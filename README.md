# Web Frontend Template

## Features
* gulp
* webpack
* eslint
* babel
* pug
* vue.js
* __No__ backend

## Installation
```
npm install
```
or
```
yarn install
cd node_modules/semantic-ui
gulp install
```
When install semantic-ui, choose Automatic -> extend current settings.
And then 
```
git checkout semantic.json
gulp init
```

## Develop
```
gulp
```
Gulp will then watch file changes and refresh the browser.
